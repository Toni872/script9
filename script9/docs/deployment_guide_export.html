Manual de Despliegue

S9-Hunter (PRACTICASFP) • v1.0

Este documento detalla cómo configurar el entorno de N8N para que el workflow funcione correctamente tras su importación.

1. Variables de Entorno (Environment Variables)

El workflow utiliza variables seguras. Debes configurarlas en la instancia de N8N (archivo .env o Cloud Config).

Variable Descripción
GOOGLE_MAPS_API_KEY Clave de API de Google (Places API activado).
ADMIN_EMAIL Email destinatario de las notificaciones.
DRIVE_PARENT_FOLDER_ID ID de la carpeta raíz de Google Drive (visible en la URL del navegador).

Configuración en Docker (.env)

# Añadir al final del archivo .env de N8N
GOOGLE_MAPS_API_KEY=AIzaSyD...
ADMIN_EMAIL=admin@cliente.com
DRIVE_PARENT_FOLDER_ID=1LbrMq6t...

2. Credenciales (Re-conexión)

Al importar el archivo JSON, las credenciales aparecerán desconectadas por seguridad. Debes re-autenticarlas.

Google Drive & Sheets

  1. Abrir nodo "Crear Carpeta Profesor".
  2. En Credential, seleccionar "Create New".
  3. Usar Google OAuth2 API y conectar cuenta corporativa.

Gmail (SMTP / OAuth2)

  1. Abrir nodo "Send a message".
  2. Crear nueva credencial Gmail OAuth2.
  3. Autorizar el envío desde la cuenta deseada.

3. Verificación Final

Antes de entregar, ejecuta una prueba manual:

  • Pulsar "Execute Workflow" y rellenar formulario de prueba.
  • Verificar que se crea la carpeta en Drive.
  • Verificar recepción del email en ADMIN_EMAIL.
  • Verificar creación de filas en Google Sheets.
